== Description ==

The Divi Masonry Gallery is a cascading grid style often used in Divi, which looks more exciting and beautiful than the basic Divi Gallery.

> Masonry Gallery, to maintain the same image width and varying heights.

## Lite features
* Easy Configuration
* Create image galleries
* Custom Columns and responsive galleries
* 100% compatible with Divi and Divi visual builder.
* Add image title and caption on hover
* Show/hide image title, show/hide image caption, change caption color and font size etc.

== Installation ==

Install the 'Divi Masonry Gallery' either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

== Frequently Asked Questions ==

= How do I install Divi Masonry Gallery? = 
To install the free version of Divi Masonry Gallery, follow the steps below:
From your WordPress dashboard -> Go to Plugins -> Click on ‘Add new’-> In the Search field, enter Divi Masonry Gallery and choose Divi Masonry Gallery.
Press Install -> After installation, click Activate.
